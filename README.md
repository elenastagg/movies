# Movies API

This API is used in this [Top 100 Movies App](https://github.com/elenastagg/top-100-movies-app) to search for and retrieve movies from [The Movies Database](https://www.themoviedb.org/).


### Prerequisites 
Docker with SLQ 

### Installing

npm install

### Running 
npm run dev
